# House Price Prediction using Machine Learning

## Project Overview
This project predicts house prices using Linear Regression.

## Dataset
- train.csv
- test.csv

## Libraries Used
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Steps Performed
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Selection
5. Train-Test Split
6. Linear Regression Model
7. Model Evaluation
8. House Price Prediction

## Model Used
Linear Regression

## Evaluation Metrics
- Mean Absolute Error
- Mean Squared Error
- R2 Score

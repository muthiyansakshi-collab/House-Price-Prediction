## Project Overview
This project analyzes house prices and provides interactive visualizations using Power BI.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Power BI

## Features
- Average Sale Price Analysis
- Maximum and Minimum Sale Price
- Year Built Analysis
- Neighborhood Analysis
- Kitchen Quality Distribution
- Garage Cars Distribution

## Files
- House_Price_Prediction.ipynb
- train.csv
- test.csv
- sample_submission.csv
- model.pkl
- requirements.txt

## Dashboard
Power BI dashboard provides interactive insights into house prices.

## Steps Performed
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Selection
5. Train-Test Split
6. Linear Regression Model
7. Model Evaluation
8. House Price Prediction

## Model Used
Linear Regression

## Evaluation Metrics
- Mean Absolute Error
- Mean Squared Error
- R2 Score
